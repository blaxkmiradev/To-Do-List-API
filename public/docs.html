Endpoints

GET /api/todos

Retrieve all to-do items. Returns an array of all tasks ordered by creation date.

Response
[ { "id": 1, "title": "Buy groceries", "description": "Milk, bread, eggs", "completed": 0, "created_at": "2024-01-01 12:00:00", "updated_at": "2024-01-01 12:00:00" } ]
GET /api/todos/:id

Get a single to-do item by its ID.

Parameter Type Description
id integer The unique identifier of the todo
Response
{ "id": 1, "title": "Buy groceries", "description": "Milk, bread, eggs", "completed": 0, "created_at": "2024-01-01 12:00:00", "updated_at": "2024-01-01 12:00:00" }
POST /api/todos

Create a new to-do item.

Parameter Type Description
title*required string The task title
description string Optional task details
Request
{ "title": "Learn Node.js", "description": "Study Express framework" }
Response
{ "id": 2, "title": "Learn Node.js", "description": "Study Express framework" }
PUT /api/todos/:id

Update an existing to-do item. All fields are optional.

Parameter Type Description
id integer The unique identifier of the todo
title string Updated task title
description string Updated description
completed integer 0 = incomplete, 1 = complete
Request
{ "title": "Updated Title", "completed": 1 }
Response
{ "message": "Todo updated", "changes": 1 }
PATCH /api/todos/:id/toggle

Toggle the completion status of a to-do item.

Parameter Type Description
id integer The unique identifier of the todo
Response
{ "message": "Todo toggled", "changes": 1 }
DELETE /api/todos/:id

Delete a to-do item permanently.

Parameter Type Description
id integer The unique identifier of the todo
Response
{ "message": "Todo deleted" }
